Estimator Classes¶

Deep Learning¶

The following deep learning models utilize TensorFlow with the Keras high-level interface. All current implementations inherit from the following abstract base class:

  • TFEstimatorBase

Current deep learning model implementations:

  • Conv6
  • HwyBLSTM (LSTM++)
  • LSTM
  • MLP

Other deep learning classes:

  • TFEpochLogger
  • HighwayLayer
