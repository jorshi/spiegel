LSTM Class¶

LSTM Deep Learning Model

Based on model proposed by Yee-King et al.

“Automatic programming of VST sound synthesizers using deep networks and other techniques.” Yee-King, Matthew John, Leon Fedden, and Mark d’Inverno. IEEE Transactions on Emerging Topics in Computational Intelligence 2.2 (2018): 150-159.

Create a tf Dataset from input and output, and shuffles / batches data for training

Parameters

  • input (np.ndarray) – matrix of training data
