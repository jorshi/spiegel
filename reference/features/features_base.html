FeaturesBase Class¶

Abstract Base Class for Audio Features

class spiegelib.features.features_base.FeaturesBase(dimensions, sample_rate=44100, frame_size=2048, hop_size=512, normalize=False, time_major=False, per_feature_normalize=True)¶

Bases: abc.ABC

Parameters

  • dimensions (int) – Number of dimensions associated with these features

  • sample_rate (int, optional) – Audio sample rate, defaults to 44100

  • frame_size (int, optional) – frame size in audio samples, defaults to 2048

  • hop_size (int, optional) – hop size in audio samples, defaults to 512

  • time_major (boolean, optional) – indicates orientation of matrix that features are returned in, time_major is (time_slices, features). Defaults to (features, time_slices).

  • per_feature_normalize (bool, optional) – Whether normalization should be applied to each dimension independently, or on entire feature space. Defaults to True, which applies normalization for each dimension independently.
